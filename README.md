# 🚀 Fun Land: Game Hub for Educators

**Fun Land** is a modern, high-fidelity platform designed for teachers to create custom interactive games for their students without any backend or database setup. Everything runs natively in the browser using **LocalStorage**, making it fast, secure, and privacy-focused.

![Fun Land Banner](https://api.qrserver.com/v1/create-qr-code/?size=150x150&data=FunLand) *(Procurar logo em breve)*

## 🎮 Game Modes

### 1. 🥇 Math Tug-of-War (Cabo de Guerra)
A competitive 1v1 math challenge where accuracy equals strength.
- **Infinite Generator**: Auto-generate equations based on Easy, Medium, or Hard presets.
- **Remote Controller Mode**: Students scan QR codes to use their phones as controllers while the arena is projected on a screen.
- **Procedural Physics**: Dynamic SVG character animations and rope tension.

### 2. 🧠 Memory Match (Jogo da Memória)
- Supports Text, Emojis, and **Custom Image Uploads**.
- Randomized layouts and professional 3D flip animations.

### 3. 🎯 Drag & Drop Match (Relacionar)
- Associate names with figures or images.
- High-performance drag-and-drop engine for an intuitive learning experience.

### 4. 📝 Custom Quiz
- Create multiple-choice questions with instant scoring.
- Visual feedback and performance tracking.

## ✨ Key Features
- **Zero Backend**: No servers needed. All data is saved directly to your browser's local storage.
- **Fully Responsive**: Play on Desktop, Tablets, or Smartphones.
- **Personalization**: Choose between 5 vibrant themes (Ocean, Forest, Fire, Sunset, Neon).
- **GitHub Pages Ready**: Optimized for static hosting with Hash Routing.
- **Teacher Onboarding**: Simple setup process for first-time users.

## 🛠️ Built With
- **React 18** (Vite)
- **Framer Motion** (Spring physics & animations)
- **Lucide React** (Modern iconography)
- **Vanilla CSS** (Custom glassmorphism design system)

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/RomarioSG1998/Fun-Land.git
cd Fun-Land
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```

### 4. Build for deployment
```bash
npm run build
```
